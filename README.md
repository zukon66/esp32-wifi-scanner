# ESP32 Wi-Fi Scanner

## Overview
ESP32 ile cevredeki Wi-Fi aglarini tarayan ve Serial Monitor'a listeleyen Arduino uygulamasi.

## Requirements
- ESP32 DevKit V1
- Arduino IDE
- `WiFi.h`

## Run
1. Arduino IDE'de `wifi_scanner.ino` dosyasini ac.
2. Board: ESP32 DevKit V1
3. Serial: 115200
4. Yukle ve Serial Monitor'da taramalari izle.

## Notes
Tarama islemi senkron calisir ve 5 saniyede bir tekrar eder.
